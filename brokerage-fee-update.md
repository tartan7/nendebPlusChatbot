# 仲介手数料の記述変更 — ウェブサイト＆チャットボット

**作成日:** 2026-07-19
**対象:** 株式会社ルーミン・ここ（roominkoko.escube.me）
**ステータス:** ドラフト（社長確認待ち）
**背景:** 社長より「宅地建物取引業界が設定している『特例』についての記述を明記してほしい」との要望。本番設置（期限2026-07-31）前に文言合意→チャット解禁と同時公開を目指す。

---

## 1. 現状の課題

### 1-1. ウェブサイト（`https://roominkoko.escube.me/aboutus/` 「02.仲介手数料」セクション）

現行の全文：

```
01 TIER 1  〜200万円  売買価格×5%＋消費税
02 TIER 2  200万〜400万円  売買価格×4%＋2万円＋消費税
03 TIER 3  400万円〜  売買価格×3%＋6万円＋消費税

上記は宅地建物取引業法に基づく上限額です。実際のご契約時はお見積りいたします。
賃貸物件の仲介手数料は別途、契約形態に応じてご案内します。
```

通常の速算法（3段階）のみで、低廉な空家等の特例には一切触れていない。稚内・宗谷エリアは空き家・低価格物件が多く、TIER1（200万円以下）の物件では通常上限（5%+税＝数万円）が現地調査コストに見合わないケースが想定される。

### 1-2. チャットボット

「仲介手数料について聞かれても『ホームページを見てください』止まりで、物件ごとの仲介手数料に答えられていない」（社長談）。

**技術的な原因を特定済み：** nendebのカスタムフィールドには物件ごとの報酬形態を持つ `houshoukeitai`（報酬形態）フィールドが存在するが、`nendeb-mcp-dify-spec.md` のmu-plugin実装（REST公開フィールド一覧、L177-193）にこのフィールドが**登録されていない**。同じ「業者情報」グループの `torihikitaiyo`（取引態様）は公開されているのに `houshoukeitai` だけ漏れている。つまりチャットボットは現状、物件ごとの報酬形態データにそもそもアクセスできない。

---

## 2. 法的根拠（確認済み）

**低廉な空家等の売買における媒介報酬の特例**（宅地建物取引業法46条・2024年7月1日施行改正）

- 対象: 売買価格**800万円以下**の空家等（土地・建物）。旧400万円以下から拡大。
- 上限: 通常の速算法上限に代えて、現地調査等の費用を勘案し合計**33万円（消費税込）**まで受領可能。
- 条件: 媒介契約締結**前**に金額を明示して依頼者に説明し、合意を得る必要がある（書面での事前説明・合意が実務上必須）。
- 2024年7月改正で、従来は主に売主側からの受領想定だったが、買主側からも受領できるよう整理された。

出典:
- https://takken-fudosan.com/01-takken-46-tokurei-20240701/
- 国交省通達: https://www.mlit.go.jp/tochi_fudousan_kensetsugyo/const/content/001749923.pdf
- 国交省通達: https://www.mlit.go.jp/tochi_fudousan_kensetsugyo/const/content/001750144.pdf

## 3. ウェブサイト文言案

既存の「02.仲介手数料」セクション、注記の直後に追加する想定。文章による説明＋具体例の表をセットで掲載する（社長の希望：「特例の場合はこうなります」を文章で明記＋例示表）。

```
低廉な空家等の売買における特例（宅地建物取引業法46条・2024年7月1日改正）

売買価格が800万円以下の低廉な空家等（土地・建物）を売買する場合、上記の上限額では
現地調査・現地訪問等に要する実費を賄いきれないケースがあります。このため宅地建物
取引業法に基づく特例として、上記の上限に代えて合計33万円（消費税込）を上限とする
仲介手数料をいただく場合があります。

この特例を適用する場合は、媒介契約の締結前に金額を明示してご説明し、ご了承をいただいた
上で契約を締結いたします。無断で特例上限を適用することはありません。

【売買価格別の上限額の例】

| 売買価格 | 通常の上限（速算法・税込） | 特例適用時の上限（税込） |
|---------|------------------------|------------------------|
| 100万円 | 5.5万円 | 33万円 |
| 300万円 | 15.4万円 | 33万円 |
| 500万円 | 23.1万円 | 33万円 |
| 800万円 | 33万円 | 33万円（＝通常上限と同額） |

※ 800万円を超える売買には本特例は適用されません（上記表の「通常の上限」のみとなります）。
※ 上表は法令上の「上限額」であり、実際にいただく金額は物件ごとに事前見積もり・ご説明の上で決定します。
```

**確認事項（社長判断）:**
- 「33万円（税込）」を具体額まで明記する方針で一旦作成（→社長判断待ち。オーナーからは「具体額まで明記して社長に判断してもらう」の方針で了承済み）。
- 例示表・文章による説明の両方を掲載する方針で確定（2026-07-19オーナー確認済み）。
- 将来法改正で金額・対象価格が変わった場合はこのページの更新が必要（現状800万円/33万円は2026-07-19時点で確認済みの最新値）。

## 4. チャットボット対応方針

**回答方針（オーナー確認済み）:** 物件ごとの `houshoukeitai` を最優先。データがなければ一般ルール（速算法＋特例）を説明。

### 4-2. Difyシステムプロンプト追加案

`dify-system-prompt.md` / `nendeb-mcp-dify-spec.md` のシステムプロンプトに以下を追加。

```
## 仲介手数料に関する質問への対応

ユーザーから仲介手数料について質問された場合：

1. 特定の物件についての質問で、その物件の houshoukeitai（報酬形態）に具体的な記載が
   ある場合は、その内容をそのまま案内する。

2. houshoukeitai が空、または一般的な質問（「仲介手数料はいくらですか？」など）の場合は、
   以下の一般ルールを説明する：
   - 売買価格400万円超：売買価格の3%＋6万円＋消費税（上限）
   - 売買価格200万円超400万円以下：売買価格の4%＋2万円＋消費税（上限）
   - 売買価格200万円以下：売買価格の5%＋消費税（上限）
   - 売買価格800万円以下の低廉な空家等は、上記に代えて合計33万円（消費税込）を上限
     とする特例が適用される場合があります。
   - 賃貸物件の仲介手数料は契約形態により異なるため、別途スタッフへの確認を案内する。

3. いずれの場合も回答の最後に必ず次を付記する：
   「正式な金額はご契約前にスタッフより書面でご説明いたします」
   （低廉な空家等の特例適用には事前の説明・合意が法律上必須のため、botの回答だけで
   金額を確定させない）
```
