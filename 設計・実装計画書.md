# 設計・実装計画書（改訂版）
## ルーミン・ココ WordPress テーマ — Gutenbergブロック化による物件検索機能実装

**プロジェクト概要**  
稚内市の不動産業者ルーミン・ココ向けWordPress 子テーマ。Home.html で設計されたUIを、SYN Ownd親テーマのGutenbergブロック体系に統合し、**ウィジェット化されたカスタムブロック** により、物件検索・フィルター・タグ検索を実装する。

**アーキテクチャの転換**
```
旧）front-page.php内でWP_Queryを直接ループ
  ↓
新）Gutenbergカスタムブロック化
  - lc-rental-properties（賃貸物件ブロック）
  - lc-sale-properties（売買物件ブロック）
  - lc-property-search（検索フォーム）
  - lc-property-tags（タグクラウド）
  → lc-top, lc-sidebar ウィジェットエリアに配置
```

**現在の状態**
- ✅ テンプレート基本構造完成（front-page.php / archive-fudo.php / single-fudo.php）
- ✅ CSS実装充実（theme.css：392行）

## 1. Gutenbergブロック化の方針

### 参考実装
- **SYN Ownd親テーマ**：`blocks/build/` でブロック.json定義、apiVersion 3
- **fudouプラグイン**：`register_block_type()` + `render_callback` + テンプレート分離

### 実装方式
1. **ブロック定義**：`block.json` 形式（`blocks/src/` に配置）
2. **登録**：`functions.php` で `register_block_type()` 
3. **テンプレート**：`render_callback` で `blocks/templates/` の PHP を include
4. **スタイル**：`blocks/css/` に block.css を配置
5. **配置**：`lc-top`（ホーム上部）/ `lc-sidebar`（サイド） ウィジェットエリアに配置

## 2. プラグイン・データ構造の確認

### 使用プラグイン
- **不動産プラグイン（fudou）**：nendeb.jp/fudou（NENDEB 公式）
- Post Type：`fudo`（物件）
- Taxonomies：
  - `fudo_kbn`（種別：賃貸/売買）
  - `fudo_tag`（タグ：設備・こだわり条件）
  - `fudo_area`（エリア）
  - ほか plugin が定義するもの

### メタデータ
- `kbn` / `shubetsu`：種別（rent / sale）
- `kakaku` / `price` / `bukken_kakaku`：価格
- `madori` / `layout`：間取り
- `menseki` / `area` / `senyu_menseki`：面積
- `shozaichi` / `address` / `jusho`：住所
- `chiku` / `chikunen` / `built_year`：築年
- `kouzou` / `structure`：構造
- ほか：`kanrihi`（管理費）、`shikikin`（敷金）、`reikin`（礼金）、`chuusha`（駐車場）、`setsubi`（設備）etc.

## 3. 実装対象ブロック

### 3.1 lc-rental-properties ブロック（賃貸物件表示）

**属性**
```json
{
  "title": "賃貸物件",
  "item": 4,
  "sort": "date",
  "displayItems": {
    "title": true,
    "price": true,
    "layout": true,

  },
  "buttonText": "物件詳細を見る"
}
```

**出力**
- `lc_render_fudo_card()` を利用、物件カード4件グリッド
- タイトル、価格、間取り、エリアを表示
- 詳細ボタンで single-fudo へ遷移

**ファイル構成**
```
blocks/src/rental-properties/
  ├── block.json
  ├── index.js (editor)
  └── render.php (template)
blocks/css/
  └── rental-properties.css
```

### 3.2 lc-sale-properties ブロック（売買物件表示）

属性・出力は lc-rental-properties と同じ（種別フィルタ `sale`）

### 3.3 lc-property-search ブロック（検索フォーム）

**属性**
```json
{
  "title": "物件を探す",
  "archiveUrl": "/archive-fudo/",
  "displayFields": ["kbn", "area", "mado", "price"]
}
```

**出力**
- Quick search フォーム（front-page.php の既存フォーム）
- `get_post_type_archive_link( 'fudo' )` へ GET パラメータ送信
- パラメータ：`kbn`, `area`, `mado`, `price_min`, `price_max`

### 3.4 lc-sidebar-search ブロック（キーワード検索フォーム）

**属性**
```json
{
  "title": "キーワードで物件を検索",
  "placeholder": "キーワードを入力"
}
```

**出力**
- キーワード検索フォーム（`get_search_form()`）
- サイドバー・アーカイブページで配置可能

### 3.5 lc-sidebar-tags ブロック（タグクラウド）

**属性**
```json
{
  "title": "設備・条件タグ",
  "taxonomy": "fudo_tag",
  "itemLimit": 20
}
```

**出力**
- タグクラウド（`get_terms( 'fudo_tag' )`）
- 各リンクは該当タームアーカイブへ

### 3.6 lc-sidebar-areas ブロック（エリアリスト）

**属性**
```json
{
  "title": "エリアから探す",
  "taxonomy": "fudo_area",
  "itemLimit": 20
}
```

**出力**
- エリアリスト（`get_terms( 'fudo_area' )`）
- 各リンクは該当タームアーカイブへ

### 3.7 lc-news-list ブロック（ニュース一覧）

**属性**
```json

  "title": "最新情報",
  "postsPerPage": 5,
```

**出力**

## 4. アーカイブページ（archive-fudo.php）の改修

### 4.1 クエリフィルター実装

**新規関数：`lc_build_fudo_query_args()`**
```php
$args = lc_build_fudo_query_args( array(
    'kbn'       => $_GET['kbn'] ?? '',
    'area'      => $_GET['area'] ?? array(),
    'mado'      => $_GET['mado'] ?? array(),
    'price_min' => $_GET['price_min'] ?? '',
    'price_max' => $_GET['price_max'] ?? '',
    'tag'       => $_GET['tag'] ?? '',
    'orderby'   => $_GET['orderby'] ?? 'date',
) );
```

**フィルターロジック**
1. **種別フィルタ** (`kbn`)：meta_query で `'kbn' == 'rent' / 'sale'`
2. **エリアフィルタ** (`area[]`)：tax_query で `fudo_area` or meta_query で複数値
3. **間取りフィルタ** (`mado[]`)：meta_query で `LIKE` 複数選択（OR結合）
4. **価格範囲** (`price_min`, `price_max`)：meta_query で `kakaku BETWEEN`
5. **タグフィルタ** (`tag`)：tax_query で `fudo_tag`
6. **並び替え** (`orderby`)：date / price_asc / price_desc

### 4.2 新規ヘルパー関数

```php
lc_parse_price( $str )          // "3万円" → 30000 に統一
lc_build_filter_chips()          // 現在の絞り込み条件を表示
lc_get_fudo_terms( $taxonomy )   // get_terms() + キャッシュ
```

### 4.3 テンプレート改修

- `archive-fudo.php` に `lc_build_fudo_query_args()` の結果を `WP_Query` に渡す
- 現在の絞り込みチップ表示ロジックを拡張
- 0件時のメッセージ表示改善
